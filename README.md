# drydocs has moved 📦→

![drydocs: put your CLAUDE.md on an intensive cardio run](assets/social-preview.png)

**drydocs now lives in the [`johanes/plugins`](https://github.com/johanes/plugins) marketplace** — a single
umbrella for all of Johanes's Claude Code plugins. Please install it from there.

## Install (new location)

```
/plugin marketplace add johanes/plugins
/plugin install drydocs@johanes
/drydocs
```

## What drydocs does

Slims a bloated `CLAUDE.md` into a **lean index** (identity + always-apply rules + pointers) and moves the
detail into a well-organized `docs/` folder that loads on demand (MOC + PARA + Zettelkasten). Non-destructive,
idempotent, git-reversible.
